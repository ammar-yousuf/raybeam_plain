Overview

There are 2 versions in the template package which are HTML and PHP versions. HTML version is the standard one and very easy to modify.

PHP version is actually similar to the HTML version except that there will be additional "template" files for some codes that need to be repeatly used on most pages like navigation menu, header, footer, CSS and JavaScript references. The template file is very useful when you would like to edit the code and need that changes to affect every page that the code is included.

Tutorial: How to Create Layout

You can easily create the layout with one simple rule: the sum of all columns in a row is 12. For example; if you created a row and you want to have 2 columns equally, you just need to create each column with the number of 6 (one column is 6 and another one is 6 then 6+6 = 12). Below is the structure for this example:

pop-uptext
  1. <div class="row">
  2.     <!-- Notice the class name: "large-6" -->
  3.     <div class="large-6 columns">
  4.         Column 1
  5.     </div>
  6.     <!-- Here is another column with 6 -->
  7.     <div class="large-6 columns">
  8.         Column 2
  9.     </div>
  10. </div>
<div class="row">
    <!-- Notice the class name: "large-6" -->
    <div class="large-6 columns">
        Column 1
    </div>
    <!-- Here is another column with 6 -->
    <div class="large-6 columns">
        Column 2
    </div>
</div>

Another example: in case you want 4 columns equally in a row, it means that you need to create each column with the number of 3 (3+3+3+3 = 12). Below is the code for this example:

pop-uptext

  2.     <!-- Notice the class name: "large-3" -->
  3.     <div class="large-3 columns">
  4.         Column 1
  5.     </div>

  13.         Column 4
  14.     </div>
  15. </div>

    <div class="large-3 columns">
        Column 1
    </div>
    <div class="large-3 columns">
        Column 2
    </div
    <div class="large-3 columns">
        Column 3
    </div>
    <div class="large-3 columns">

From the above code, there are 2 nested columns which are Image Column and Text Column inside the first half column. Now, let's see the detail on Image Column.

Image Column contains 2 optional classes which are height-510 and no-padding. height-510 is for forcing the column to have 510px height. no-padding is for removing all padding of the column so the contained image will fill the entire area of the column. The image tag itself contains stretch-image class. The class is for maintaining the ratio of the image on smaller resolution like iPad (the image will be stretched filling the column).

The next one is Text Column. This column is for title, description and button. There are 2 optional classes which are height-510 and bottom-line. height-510 is for forcing the column to have 510px height. bottom-line is for displaying the line at the bottom of the column.

Change Template Color

Archtek comes with 6 colors which are blue (default), green, red, orange, yellow and purple. To change the color, please open the file "/css/archtek-init.css". Then just follow the instruction mentioned in the file.

Change Background Patterns

You can change the background from solid color to image pattern by editing the CSS selectors in "/css/archtek.css" file. Related selectors are body and .testimonial-corner. Note that Archtek comes with some sample patterns which are in "/images/bg" folder. If you would like to find more, please go to http://subtlepatterns.com/.

Configure Google Map Element

Basically, Google Map element has the structure as following:

pop-uptext
  1. <div class="google-map" data-latlng="" data-address="" data-display-type="" data-zoom-level="" data-height=""></div>
<div class="google-map" data-latlng="" data-address="" data-display-type="" data-zoom-level="" data-height=""></div>

Here is the explanation for each attribute:

  • class: This must be "google-map"
  • data-latlng: Latitude and logitude value. The template will use this pair for generating the map only if "data-address" below is left blank. To find your latitude and longitude, please go to http://itouchmap.com/latlong.html
  • data-address: The template will use this as the primary value for generating the map. If you leave it blank, "data-latlng" will be used instead.
  • data-display-type: You can choose which display type to use (ROADMAP, SATELLITE, HYBRID, TERRAIN).
  • data-zoom-level: Specify a number for zoom level (0-18).
  • data-height: The height of the map in pixel.

Before the contact form can send the submitted data to your email inbox, you would need to configure some values of PHPMailer first. They are mostly related to your SMTP server information. Please open "/php/sendmail.php" file and here are the important values that you have to edit:

pop-uptext
  1. $site_owners_email = 'your@mail.com'; // Replace this with your own email address
  2. $site_owners_name = 'uxbarn'; // Replace with your name
  3. $mail->Host = "smtp.yoursite.com"; // Replace with your SMTP server address
  4. $mail->Username = "user@smtp.com"; // SMTP username
  5. $mail->Password = "yourpassword"; // SMTP password
